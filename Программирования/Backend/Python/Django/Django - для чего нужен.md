- Django — это Python web-фреймворк для быстрого создания сайтов и API.
- Принцип: "Batteries included" (много всего встроено).
- Основные возможности:
  - ORM (работа с БД без SQL)
  - Админ-панель "из коробки"
  - Система аутентификации
  - Маршрутизация URL
  - Работа с шаблонами
  - DRF (Django Rest Framework) для API

## Когда использовать?
- Когда нужен **быстрый старт** (блог, сайт, магазин, API).
- Когда важна **безопасность** (авторизация, CSRF, XSS).
- Когда проект будет **масштабироваться**.
- Django монолитный фраемворк
