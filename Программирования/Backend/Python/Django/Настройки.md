**Middleware** — это “промежуточный слой”, который обрабатывает **каждый HTTP-запрос и ответ** между браузером (клиентом) и твоим Django-приложением.

Проще: это как **перехватчик**, который может что-то сделать **до** того, как запрос попадёт в View, и/или **после** того, как View вернёт ответ.

**WSGI** = **Web Server Gateway Interface**  
Это стандарт Python для взаимодействия веб-сервера (например, Gunicorn, uWSGI) с твоим Django-приложением.

> WSGI_APPLICATION = 'sim.wsgi.application'

**ASGI** = **Asynchronous Server Gateway Interface**  
Это современный стандарт Python для **асинхронных приложений**.
> 
> ASGI_APPLICATION = 'sim.asgi.application'

  

`CHANNEL_LAYERS` — это **конфигурация слоя каналов**, который нужен для обмена сообщениями между разными частями приложения, особенно для **WebSocket, асинхронных задач, групповых чатов** и **уведомлений в реальном времени**.

> CHANNEL_LAYERS = {
>     "default": {"BACKEND": "channels.layers.InMemoryChannelLayer"},
> 
> }
> 
  
  


  `DATABASES` — это словарь в `settings.py`, который **указывает Django, с какой базой данных работать** и как к ней подключаться.

По сути, это **настройка подключения к БД** для ORM (Object-Relational Mapping).

1. Sqlite (по умолчанию)
 DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': BASE_DIR / 'db.sqlite3',
    }
}

  
2. PostgreSQL
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        "NAME": os.getenv("DB_NAME"),
        "USER": os.getenv("DB_USER"),
        "PASSWORD": os.getenv("DB_PASSWORD"),
        "HOST": os.getenv("DB_HOST"),
        "PORT": os.getenv("DB_PORT"),
    }
}

- Определяет **язык интерфейса Django**.
> LANGUAGE_CODE = 'ru-ru'
- Задаёт **часовой пояс проекта**.
> TIME_ZONE = 'Asia/Bishkek'

- **I18N = Internationalization (интернационализация)**
    
- `True` значит, что Django будет использовать механизмы перевода и мультиязычности
> USE_I18N = True

- Включает поддержку **часовых поясов**.
> USE_TZ = True

- Указывает **URL страницы входа**, куда перенаправляют пользователя, если он не авторизован.
> LOGIN_URL = reverse_lazy('users:login')
- Нужно для **Django sites framework** (если у тебя один сайт, оставь 1).
> SITE_ID = 1

- URL, по которому браузер будет получать **статические файлы** (CSS, JS, картинки).
STATIC_URL = '/static/'
 # это куда Django будет копировать статику

- Папка, куда Django будет **собирать все статические файлы** при деплое (`collectstatic`).
STATIC_ROOT = os.path.join(BASE_DIR, 'staticfiles')

`STATICFILES_DIRS` — папки с твоими статическими файлами **во время разработки**.
В продакшене все файлы собираются в `STATIC_ROOT`.


```
if DEBUG:
    STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]
else:
    STATIC_ROOT = os.path.join(BASE_DIR, 'staticfiles')
```
