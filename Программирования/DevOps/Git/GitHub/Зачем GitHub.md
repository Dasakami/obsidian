GitHub — **это облачная платформа для хостинга IT-проектов и совместной разработки**. Под её капотом находится популярная система контроля версий Git. 

**Зачем GitHub нужен**:

- **Упрощение задач при работе над большими проектами**. Можно отслеживать изменения, при ошибке — откатить к предыдущей версии или сотрудничать с другими разработчиками.
- **Централизованное хранение кода проекта и его документации**. Есть лёгкий доступ ко всем файлам и возможность свободно поделиться с новыми участниками.
- **Социальная платформа**. Разработчики могут показывать свои проекты, вносить вклад в чужую работу и строить свою профессиональную репутацию. При желании можно создать портфолио своих проектов и достижений.



**Что делает GitHub**:

- предлагает безопасную авторизацию по токенам;
- предлагает инструменты тестирования и анализа кода;
- предлагает сервисы деплоя проектов (GitHub Actions).
