## 1. компакт-диск

Команда **cd (Изменить каталог)** в Linux — одна из самых важных и широко используемых команд. Он используется для изменения текущего рабочего каталога.

Перейдите из текущего каталога в `/tmp`.

```dos
cd /tmp
```

Вернитесь в предыдущий каталог, в котором вы работали ранее.

```dos
cd -
```

Измените текущий каталог на родительский.

```dos
cd ..
```

Перейдите в домашний каталог пользователя из любого места.

```dos
cd
```

## 3. лс

**ls (список файлов и каталогов)** – одна из основных команд, которую должен знать любой пользователь Linux. В нем перечислено содержимое каталога, например файлы и папки.

Запуск `ls` без параметров выведет содержимое текущего каталога.

```powershell
ls
```

```undefined
psforevermore.txt  pulse-linux-9.1r2.0-x64.rpm  website-logo.jpg
```

Использование параметра `-l` (длинный формат) отобразит длинный список содержимого текущего каталога. Команда выведет не только имя файла, но и некоторые атрибуты, такие как:

- разрешения
- владелец
- владелец группы
- размер файла в байтах
- время и дата изменения файла.

```powershell
ls -l
```

```xml

-rw-r--r-- 1 <meta http-equiv="content-type" content="text/html; charset=utf-8">linuxiac <meta http-equiv="content-type" content="text/html; charset=utf-8">linuxiac 23271352 Jul 28 14:57 pulse-linux-9.1r2.0-x64.rpm

Чтобы просмотреть содержимое определенного каталога, используйте команду ниже.

```powershell

```xml
<meta http-equiv="content-type" content="text/html; charset=utf-8">total 22968

С опцией `-h` `ls` отобразит размеры файлов в удобочитаемом формате. Этот параметр имеет смысл только в сочетании с параметром `-l` .

В Linux файл, начинающийся с `.` , является скрытым файлом. Чтобы отобразить его в команде `ls`, мы можем использовать параметр `-a` .

## 4. КП

**cp (Копировать)** — это команда, используемая для копирования файлов и каталогов в Linux. Чтобы скопировать файл с помощью команды `cp` , передайте имя файла, который нужно скопировать, а затем место назначения.

Например, выполнение приведенной ниже команды скопирует файл `website-logo.jpg` в каталог `/tmp/`.

```powershell
cp website-logo.jpg /tmp/
```

Вам необходимо указать желаемое имя файла, если вы хотите скопировать файл под другим именем.

```powershell
<meta http-equiv="content-type" content="text/html; charset=utf-8">cp website-logo.jpg /tmp/new-logo.jpg
```

Чтобы скопировать несколько файлов, передайте имена файлов, а затем каталог назначения, команде `cp` .

```powershell
cp website-logo.jpg psforevermore.txt images/
```

Чтобы скопировать каталог, включая все его файлы и подкаталоги, используйте параметр `-r` (рекурсивный).

Например, мы копируем каталог `images` в `images_bckp`.

```powershell
cp -r images/ images_bckp/
```

## 5. МВ

**mv (Переместить)** используется для перемещения одного или нескольких файлов или каталогов из одного места в другое. Помимо перемещения файлов, он также может переименовать файл или каталог.

Например, чтобы переместить файл с именем `website-logo.jpg` из текущего каталога в в каталоге `images` команда будет такой:

```powershell
mv website-logo.jpg images/
```

Если вы хотите переименовать файл с именем `website-logo.jpg` в `new -logo.jpg`, вы можете использовать команду `mv` следующим образом:

```powershell
<meta http-equiv="content-type" content="text/html; charset=utf-8">mv website-logo.jpg new-logo.jpg
```

Как и при переименовании файла, вы можете переименовать каталог с помощью команды `mv`.

Например, чтобы переименовать каталог с именем `images` в `images_bckp`, команда будет такой:

```powershell
mv images images_bckp
```

## 6. РМ

**rm (Удалить)** – это утилита командной строки для удаления файлов и каталогов.

**По теме: Как удалить файлы и каталоги в Linux из командной строки**

Чтобы удалить один файл, используйте команду `rm` , за которой следует имя файла в качестве аргумента:

```powershell
rm website-logo.jpg
```

По умолчанию `rm` не удаляет каталоги. Однако если указана опция `-r` (рекурсивная), `rm` удалит все соответствующие каталоги и их содержимое.

```powershell
rm -r images/
```

Команда `rm` запросит подтверждение, защищен ли данный каталог или файл в нем от записи.

Чтобы удалить каталог с именем `images` без запроса, используйте параметр `-f` :

```powershell
rm -rf images/
```

**Внимание**. Будьте осторожны при выполнении команды `rm -rf` . Небольшая опечатка или незнание **может привести к необратимому повреждению системы**.

## 7. мкдир

**mkdir (Создать каталог)** – это основная команда Linux для создания каталога. Команда `rm` позволяет удалять каталоги, а команда `mkdir` позволяет их создавать.

Создать каталоги довольно просто. Все, что вам нужно сделать, это передать имя каталога, который вы хотите создать, команде `mkdir`.

Например, чтобы создать новый каталог с именем `images` в текущем каталоге, команда будет быть:

```dos
mkdir images
```

Для создания структуры с несколькими подкаталогами с помощью `mkdir` требуется добавить параметр `-p` . Опция `-p` указывает `mkdir` также создать родительские каталоги.

```dos
mkdir -p images/upload/new
```

## 12. Греп

**grep (Global Regular Expression Print)** ищет строку символов в указанном файле. Это одна из самых полезных команд в Linux.

Например, чтобы выполнить поиск в любой строке, содержащей слово `color` в имени файла `psforevermore.txt`, команда будет:

```dos
grep color psforevermore.com
```

```xml
You're the <strong>color</strong> of my life,
```

Поскольку команда `grep` чувствительна к регистру, одним из наиболее полезных операторов для поиска `grep` является `-i`. Вместо того, чтобы печатать только результаты в нижнем регистре, терминал отображает результаты как в верхнем, так и в нижнем регистре.

```dos
grep -i Color psforevermore.txt
```

```xml
<meta http-equiv="content-type" content="text/html; charset=utf-8">You're the <strong>color</strong> of my life,
```

Чтобы включить в поиск все подкаталоги, добавьте оператор `-r` (рекурсивно) к команде `grep`.

Приведенный ниже пример команды печатает совпадения для слова `username` во всех файлах в каталоге `/etc` и его подкаталогах.

```undefined
grep -r username /etc
```

## 14. чаун

Команда **`chown` (Изменить владельца)** используется для изменения владельца или группы файла. Всякий раз, когда вы хотите сменить владельца, вы можете использовать команду `chown`. Права суперпользователя (`sudo`) необходимы для выполнения команды `chown`.

Для изменения владельца файла с помощью `chown` вам необходимо указать нового владельца и файл.

Следующая команда меняет владельца файла `website-logo.jpg` с `linuxiac` на пользователя `john`:

```undefined
chown john website-logo.jpg
```

Чтобы назначить нового владельца файла и одновременно изменить его группу, выполните команду `chown` в формате, указанном ниже.

Например, чтобы установить `john` в качестве нового владельца и `users` в качестве новой группы файла `website-logo.jpg`:

```undefined
chown john:users website-logo.jpg
```

Команда `chown` позволяет изменить владельца всех файлов и подкаталогов в указанном каталоге. Для этого добавьте в команду параметр `-R`.

В следующем примере мы рекурсивно изменим владельца и группу для всех файлов и каталогов в каталоге `images`.

```undefined
chown -R john:users images/
```

## 15. чмод

**chmod (режим изменения)** позволяет изменить права доступа к файлу. Только root, владелец файла или пользователь с привилегиями `sudo` могут изменять разрешения файла.

Ссылки используются для различения пользователей, к которым применяются разрешения:

- **`u` (владелец)**: владелец файла.
- **`g` (группа)**: пользователи, являющиеся членами группы файла.
- **`o` (другие)**: пользователи, которые не являются ни владельцами файла, ни членами его группы.
- **`a` (все)**: все три из вышеперечисленных (то же, что `ugo`).

Операторы используются для указания того, как следует настраивать режимы файла:

- ```undefined
    +
    ```
    

Добавляет указанные режимы к указанным классам.

- ```undefined
    -
    ```
    

Удаляет указанные режимы из указанных классов.

- ```undefined
    =
    ```
    

Указанные режимы должны быть точными режимами для указанных классов.

Режимы указывают, какие разрешения следует предоставить или удалить из указанных классов:

- ```powershell
    r
    ```
    

Прочтите файл.

- ```undefined
    w
    ```
    

Запишите или удалите файл.

- `**x**` Выполните файл или, в случае каталога, выполните поиск в нем.

Например, в следующей команде разрешение на чтение будет добавлено для всех трех уровней: пользователя, группы и другого.

```powershell
chmod a+r <meta http-equiv="content-type" content="text/html; charset=utf-8">website-logo.jpg
```

Удалите разрешение на выполнение для всех пользователей:

```xml
chmod a-x <meta http-equiv="content-type" content="text/html; charset=utf-8">website-logo.jpg
```

Удалите разрешения на чтение, запись и выполнение для всех пользователей, кроме владельца файла:

```xml
chmod og-rwx <meta http-equiv="content-type" content="text/html; charset=utf-8">website-logo.jpg
```

Если вы хотите, чтобы владелец файла имел права на чтение и запись, а группа и другие пользователи имели права только на чтение:

```powershell
chmod u=rw,go=r <meta http-equiv="content-type" content="text/html; charset=utf-8">website-logo.jpg
```

Использование оператора **=** означает, что мы удаляем существующие разрешения, а затем устанавливаем указанные.

Вы можете добавить разрешение на выполнение для всех с помощью следующей команды:

```xml
chmod a+x <meta http-equiv="content-type" content="text/html; charset=utf-8">website-logo.jpg
```

Чтобы рекурсивно работать со всеми файлами и каталогами в заданном каталоге, используйте команду `chmod` с опцией `-R` (рекурсивно).

Например, в следующей команде пользователь может читать, писать и выполнять. Однако члены группы и другие пользователи могут читать и выполнять операции, но не могут писать. Это относится ко всем всем файлам и подкаталогам в каталоге `/tmp/test` .

```undefined
chmod -R u=rwx,go=rx /tmp/test
```

## 17. найти

Команда **найти** позволяет пользователям искать файлы и каталоги по условиям. Это одна из самых важных и часто используемых команд в Linux.

Найдите все файлы с именем `website-logo.jpg` в текущем рабочем каталоге.

```dos
find . -name website-logo.jpg
```

Параметр `-name` чувствителен к регистру. Если вы не знаете точный регистр искомого элемента, вы можете использовать параметр `-iname` , который не чувствителен к регистру.

```xml
<meta http-equiv="content-type" content="text/html; charset=utf-8">find . -iname website-logo.jpg
```

Найдите все каталоги с именем `linuxiac` в `/` каталоге.

```dos
find / -type d -name linuxiac
```

Найдите все файлы `.php` в каталоге `/var/www/html`.

```dos
find /var/www/html -type f -name "*.php"
```

## 20. пс

Команда **`ps` (Состояние процесса)** отображает текущие процессы в системе. Кроме того, он отобразит список процессов, запущенных в системе, включая такие детали, как идентификатор процесса, имя терминала, на котором в данный момент выполнен вход, время процессора и т. д.

Системные администраторы обычно используют ps с опциями `a`, `u`, `x` и `w`, чтобы получить все подробности в одном файле. ps выполнение команды.

- ```undefined
    a
    ```
    

: Все остальные пользовательские процессы.

- ```undefined
    u
    ```
    

: Владелец процесса.

- ```undefined
    x
    ```
    

: Другие процессы, не подключенные к терминалу.

- ```undefined
    w
    ```
    

: Широкий выходной сигнал.

```powershell
ps auxw
```
