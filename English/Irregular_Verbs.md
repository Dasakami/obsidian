# Irregular Verbs (Неправильные глаголы)

Подробное руководство и полный список находится в:

→ **[[07_Irregular_Verbs/]]** — с разделением по группам

Неправильные глаголы — это глаголы, которые не следуют стандартной схеме образования прошедшего времени (V + -ed).

**Примеры**:
- go → went → gone (не "goed")
- be → was/were → been (не "beed")
- eat → ate → eaten (не "eated")
- see → saw → seen (не "seeped")

**Группы**:
- **Group 1**: Нет изменений (put → put → put)
- **Group 2**: Изменение гласной (sing → sang → sung)
- **Group 3**: V2 = V3 (cut → cut → cut)
- **Group 4**: Полное изменение (go → went → gone)
