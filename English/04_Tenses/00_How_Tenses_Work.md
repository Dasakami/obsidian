# How Tenses Work

## Core Idea

Tense = 2 things:

1. **TIME** — когда происходит действие
    
2. **FORM** — как строится предложение
    

> 90% ошибок — это **форма**, а не время.

---

## Every tense has

- **Statement (+)** — утверждение
    
- **Question (?)** — вопрос
    
- **Negative (-)** — отрицание
    

Questions and negatives depend on **auxiliary verbs** (do / does / did / will / have / is / etc.)

---

## Key Tips

- Focus on **FORM first**, TIME comes naturally.
    
- If you know **which auxiliary to use**, you can make **any question or negative**.
    
- Verb form after auxiliary → **always base form** (V1), except for continuous/perfect tenses where the verb changes.
    

---

## Visual Idea (for your brain)

```
TENSE = TIME + FORM
FORM = + / ? / -
Questions → auxiliary + base verb
Negatives → auxiliary + not + base verb
```
