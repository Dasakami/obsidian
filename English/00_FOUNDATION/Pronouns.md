# Pronouns (Местоимения)

## Определение | Definition

**Местоимения** - слова, которые заменяют существительное, чтобы избежать повторений.

**Pronouns** are words that replace nouns to avoid repetition.

---

## Типы местоимений | Types of Pronouns

### 1. Personal Pronouns (Личные местоимения)

#### Subject Form (подлежащее)
```
I       (я)
You     (ты/вы)
He      (он)
She     (она)
It      (оно)
We      (мы)
They    (они)
```

**Пример**: **I** like coffee.

#### Object Form (дополнение)
```
me      (мне/меня)
you     (тебе/тебя/вам)
him     (ему/его)
her     (ей/её)
it      (ему/его)
us      (нам/нас)
them    (им/их)
```

**Пример**: She likes **me**.

---

### 2. Possessive Pronouns (Притяжательные местоимения)

| | |
|---|---|
| mine | (мой) |
| yours | (твой/ваш) |
| his | (его) |
| hers | (её) |
| its | (его/её - для вещей) |
| ours | (наш) |
| theirs | (их) |

**Пример**: This book is **mine**. (Эта книга моя.)

---

### 3. Possessive Adjectives (Притяжательные прилагательные)

| | |
|---|---|
| my | (мой) |
| your | (твой/ваш) |
| his | (его) |
| her | (её) |
| its | (его/её) |
| our | (наш) |
| their | (их) |

**Пример**: This is **my** book. (Это моя книга.)

**Отличие**: 
- Possessive pronoun: "This book is **mine**" (самостоятельное)
- Possessive adjective: "This is **my** book" (перед существительным)

---

### 4. Reflexive Pronouns (Возвратные местоимения)

```
myself       (сам/сама)
yourself     (сам/сама)
himself      (сам)
herself      (сама)
itself       (самого себя)
ourselves    (сами)
yourselves   (сами)
themselves   (сами)
```

**Примеры**:
- "I did it **myself**" (Я сделал это сам)
- "She hurt **herself**" (Она ушибла себя)
- "They enjoyed **themselves**" (Они хорошо провели время)

---

### 5. Demonstrative Pronouns (Указательные местоимения)

| | |
|---|---|
| this | (этот - близко) |
| that | (тот - далеко) |
| these | (эти - близко) |
| those | (те - далеко) |

**Примеры**:
- "**This** is interesting" (Это интересно)
- "**That** is boring" (То скучно)

---

### 6. Interrogative Pronouns (Вопросительные местоимения)

| | |
|---|---|
| who | (кто?) |
| whom | (кого?) |
| whose | (чей?) |
| what | (что?) |
| which | (какой?) |

**Примеры**:
- "**Who** is it?" (Кто это?)
- "**What** do you want?" (Что ты хочешь?)
- "**Which** one?" (Какой из них?)
