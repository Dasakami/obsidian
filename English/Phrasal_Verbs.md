# Phrasal Verbs (Фразовые глаголы)

Подробное руководство и списки находятся в:

→ **[[09_Phrasal_Verbs/]]** — с примерами и категориями

Фразовые глаголы — это комбинация глагола + предлога/наречия, которые создают новое значение.

**Примеры**:
- **get up** = проснуться (не "получить вверх")
- **put on** = надеть (не "положить на")
- **look after** = заботиться (не "смотреть после")
- **break down** = сломаться (не "сломать вниз")

## Структура

```
Verb + Particle (Preposition/Adverb) = New Meaning

run + away = убегать
turn + on = включить
look + for = искать
```

---

## Почему они важны?

⭐ 100+ фразовых глаголов охватывают большую часть разговорной речи
⭐ Часто встречаются в речи носителей
⭐ Трудно угадать значение из отдельных слов
