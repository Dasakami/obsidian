# Action Verbs (Глаголы действия)

**Глаголы действия** — выражают активное действие, которое можно видеть или слышать.

## Характеристики

✅ Можно использовать во **все** аспекты (Simple, Continuous, Perfect)
✅ Имеют активный характер
✅ Отвечают на вопрос "Что делать?"

## Примеры

### Физические действия
- run (бегать)
- jump (прыгать)
- eat (есть)
- write (писать)
- build (строить)
- dance (танцевать)

### Речевые действия
- speak (говорить)
- tell (рассказывать)
- ask (спрашивать)
- answer (отвечать)

### Умственные действия (с видимым результатом)
- study (учить)
- learn (учиться)
- solve (решать)
- plan (планировать)

---

## Примеры в разных временах

- **Present Simple**: "I **run** every day"
- **Present Continuous**: "I **am running** now"
- **Present Perfect**: "I **have run** 5 km today"
- **Past Simple**: "I **ran** yesterday"
- **Past Continuous**: "I **was running** when you called"

---

## Отличие от State Verbs

❌ State Verbs нельзя использовать в Continuous
✅ Action Verbs можно использовать во все времена
