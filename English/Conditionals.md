# Conditionals (Условные предложения)

Подробное руководство находится в:

→ **[[05_Constructions/10_Conditionals_0_1.md]]** — Type 0, 1 (реальные условия)
→ **[[05_Constructions/11_Conditionals_2_3.md]]** — Type 2, 3 (нереальные условия)

Условные предложения используют **если... то...** структуру для выражения зависимостей и предположений.

**Быстро**:
- **Type 0**: "If you heat water, it boils" (закон природы)
- **Type 1**: "If it rains, I will stay home" (вероятно произойдёт)
- **Type 2**: "If I were rich, I would travel" (воображаемое)
- **Type 3**: "If I had known, I would have told you" (прошлое, которое не произошло)

**Откройте** соответствующие файлы для полного объяснения, примеров и практики.
